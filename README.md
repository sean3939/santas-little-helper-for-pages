# Santa's Little Helper (The Santa Scanner)

This is the version used to hose on Github pages. Minor tweaks needed to get it up and running.

This is a **Serverless Client-Side Application** built with React, Vite, and the Google GenAI SDK. It relies entirely on the browser to orchestrate AI agents.

### The Architecture
1.  **The Elves (Data Gathering)**:
    -   **News**: Uses `gemini-2.5-pro` with **Google Search Grounding** to perform a live background check on any public figure.
    -   **GitHub**: Fetches raw user activity via the GitHub REST API to judge code quality and commit habits.
    -   **Reddit**: Attempts direct JSON fetching, falling back to Gemini Research if blocked by CORS.
    -   **Wikipedia**: Scrapes biographical data from the MediaWiki API.

2.  **The Brain (Judgment)**:
    -   **Model**: `gemini-2.5-pro`
    -   **Role**: Analyzes the raw text/JSON from the gathering phase. It operates on a strict System Instruction to output a structured JSON verdict including a "Naughty" or "Nice" rating (0-100), a witty persona title, and a rationale in Santa's voice.

3.  **The Toy Shop (Visualization)**:
    -   **Model**: `gemini-2.5-flash-image` (Internal Codename: **Nano Banana**)
    -   **Role**: Takes the `visual_prompt` generated by The Brain and creates a high-quality, 1:1 aspect ratio image of the specific gift or punishment (e.g., "A lump of coal with a 404 error carved into it").

4.  **The Concierge (Shopping)**:
    -   If a user scores >40, The Brain also generates specific real-world gift recommendations, which the UI links directly to Google Shopping.
